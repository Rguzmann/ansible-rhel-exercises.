# Ansible RHEL Exercise Library

This repository contains workshop exercises and templates for creating new labs for Ansible on Red Hat Enterprise Linux.

The goal is to provide:

- Ready to use exercises you can drop into workshops or demos  
- A standard template for colleagues who want to create new exercises  
- A central place to maintain and update content
